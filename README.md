This project implements a simplified version of the **BZip2 compression algorithm** as part of a course assignment. The implementation is structured in multiple stages, with each stage incrementally building the full compression pipeline.

Stage 1 focuses on the foundational components required for compression, including file handling, block processing, and core transformation algorithms.
